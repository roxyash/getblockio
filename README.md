# weatherchecker

### Project setup
## install util make
Windows:
```
scoop install make
```

Macos:
```
brew install make
```

### Compiles and hot-reloads for development with docker
```
make getblock
```

### Compiles and hot-reloads for development without docker
```
make run
```

### After refactor swagger docs
```
make swag
```
